AR Namecard App

An augmented reality namecard experience that brings personal branding to life in 3D.

Project Overview

An augmented reality namecard using Unity and Vuforia that displays contact details and a 3D slideshow of projects when scanned.

Key Features

  • AR marker detection
  • Interactive slideshow
  • Personal branding integration

Technical Highlights

Developed in Unity with Vuforia and C#; custom visuals designed in Illustrator.

Impact

Blended branding with interactive technology to create a memorable AR experience.

Technologies Used

Unity C# Adobe Illustrator

Project Details

Duration: 4 months

Team Size: Solo project
