VR Escape Room

An immersive virtual reality game designed in Unity with interactive puzzle mechanics.

Project Overview

A VR escape room made using Unity together with C# scripts, this was made as part of my graded assignment for my Immersive Technologies module in Republic Polytechnic.

Key Features

  • Fully interactive VR environment
  • Puzzle-based progression system
  • Gesture and gaze-based interaction
  • 3D audio and visuals for immersion

Technical Highlights

Created with Unity and C#, using coroutines and object interaction scripting.

Impact

Enhanced skills in VR development and interactive game design.

Technologies Used

Unity C#

Project Details

Duration: 6 months

Team Size: 1 developer
